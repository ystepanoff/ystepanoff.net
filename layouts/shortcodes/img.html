{{- $imagePath := .Get "path" -}} {{- $altText := .Get "alt" | default "Image" -}} {{- $image := resources.Get $imagePath -}} {{- if $image -}} {{- /* Cap the rendered width, downscale large originals, and serve WebP. */ -}} {{- $maxWidth := 1000 -}} {{- if gt $image.Width $maxWidth -}} {{- $image = $image.Resize (printf "%dx webp q85" $maxWidth) -}} {{- else -}} {{- $image = $image.Process "webp q85" -}} {{- end -}} {{- else -}} {{- errorf "img shortcode: image not found at path %q" $imagePath -}} {{- end -}}
